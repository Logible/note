# 用于将文字加上颜色

$\color{#4285f4}{Shit}$
 <!--红色  -->
$\color{#ea4335}{Shit}$
<!-- 黄色 -->
$\color{#fbbc05}{Shit}$
<!-- 绿色 -->
$\color{#34a853}{Shit}$
<!-- 紫色 -->
$\color{#B620E0}{Shit}$
<!-- 红色加强 -->
<code style="color:#ea4335">RedShit</code>
<!-- 黄色警告 -->
<code style="color:#fbbc05">YellowShit</code>
<!-- 紫色疑惑 -->
<code style="color:#B620E0">YellowShit</code>
