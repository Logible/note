# Functions

## unsqueeze

```py
x = torch.tensor([1, 2, 3, 4])
print(x)
# unsqueeze 函数在第一个维度 (dim=0) 上插入一个新的维度
print(torch.unsqueeze(x, dim=0))
# unsqueeze 函数在第二个维度 (dim=1) 上插入一个新的维度
print(torch.unsqueeze(x, dim=1))
```
