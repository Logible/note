# 模板

- 解压图片
  - ffmpeg -i ⭐.mp4 -vf "select=not(mod(n\,1))" -vsync 0 ⭐/%05d.png

- 抠图
  - python main.py --img_dir ⭐ --out_dir ⭐ --color cg --bg ⭐ --multi_process 20 --mask_th ⭐

- 合成视频
  - python get_seg_video3.py ---path ⭐/ --video_file ⭐.mp4 --start_id 1 --end_id -1

- 删除文件
  - rm -rf ./⭐/*
